FCSTN UNITY PROJECT — SETUP GUIDE
====================================

Prerequisites:
  1. Unity Hub (already installed)
  2. Unity Editor 2022.3 LTS — install via Unity Hub

Steps:
  1. Open Unity Hub
  2. Click "Open" → select this folder (FCSTNProject)
  3. If prompted, install Unity 2022.3.20f1 (or any 2022.3 LTS)
  4. Once the project opens, go to Assets/Scenes and create a new scene (File → Save As → MainScene)

Create the scene:
  1. Right-click in Hierarchy → Create Empty → name it "GameManager"
  2. Select GameManager → Add Component → FCSTNClient
  3. Create objects and drag them into the FCSTNClient slots:
     - fractalMesh: Create a 3D Object → Sphere (or import your fractal mesh)
     - mainCamera: Main Camera (already exists)
     - sceneLight: Directional Light (already exists)
     - fractalParticles: Create → Effects → Particle System (name: "FractalParticles")
       - Enable Noise module in the Inspector
     - glitchParticles: Create → Effects → Particle System (name: "GlitchParticles")
       - Set start color to red
  4. Create Glitch Material:
     - In Assets/Materials, right-click → Create → Material (name: "GlitchMat")
     - In the Inspector, set Shader to "Hidden/FCSTNGlitch"
     - Assign GlitchMat to FCSTNClient.glitchMaterial
  5. Press Play — the client connects to ws://localhost:8765

To connect to your running FCSTN server:
  - Start the server: python deploy_server.py
  - Or start the FCSTN server: python -m server.fcstn_server
  - The Unity client auto-connects and starts receiving metrics

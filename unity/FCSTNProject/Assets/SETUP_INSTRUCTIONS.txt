FCSTN UNITY PROJECT — STANDALONE DEMO
======================================

Prerequisites:
  1. Unity Hub
  2. Unity Editor 2022.3 LTS (2022.3.20f1) — install via Unity Hub

Quick Start (2 minutes):
  1. Open Unity Hub → Open → select this folder (FCSTNProject)
  2. If prompted, install Unity 2022.3.20f1
  3. Once the project loads, in the Project window go to Assets/Scenes/
  4. Open MainScene.unity (if it exists) OR create a new scene:
     File → New Scene → Save As → Assets/Scenes/MainScene.unity
  5. In the Hierarchy, right-click → Create Empty → name it "GameManager"
  6. Select GameManager → Add Component → search for "FCSTNClient"
  7. Press Play — the demo starts immediately with synthetic cognitive data!

No server setup needed — the FCSTNClient auto-creates all objects
(sphere mesh, particles, lights, line renderer, glitch material) and
generates realistic cognitive data in demo mode.

Controls (in the Inspector with GameManager selected):
  - demoMode: toggle ON (default) for standalone; OFF to connect to Python server
  - demoSpeed: adjust simulation speed (0.1–5.0)
  - glitchMaterial: assign a material using the "Hidden/FCSTNGlitch" shader
    for post-processing effects (auto-generated if left unassigned)

Live Server Mode (optional):
  - Set demoMode = false
  - Start the FCSTN Python server: python -m server.fcstn_server
  - The client connects to ws://localhost:8765 and receives real BCI data

Motor Fractal

z(n+1) = z(n)^2 + c
R = 2, L = 64..1024
D = -lim log(N(ε)) / log(ε)
Smooth coloring:
μ = n + 1 - log(log|z(n)|)/log(2)

Dimensión fractal D modula el tensor métrico y la complejidad del mapa de juego.

D < 2.0: Minimalista, suave

D ~ 2.5: Clásico Mandelbrot

D > 3.0: Hiper-complejo, caótico

Tensor Métrico

Embedding del campo fractal en un espacio-tiempo 4D pseudo-Riemanniano

Minkowski (plano)

g = diag(-1, a^2, a^2, a^2)
a = 1 + D · α
α = índice de atención

Schwarzschild (curvo)

g_tt = -(1 - rs/r)
g_rr = (1 - rs/r)^(-1)
rs = f(engagement)

La curvatura del espacio-tiempo refleja el estado cognitivo del usuario en tiempo real.

FCSTN: Fractal Cognitive Spacetime Network

Fernando May1, Diego Armando Linares2
1 Instituto Politécnico Nacional, ESCOM, Mexico City
2 Universidad Autónoma Metropolitana, Cuajimalpa
Correspondence: fernando.may@fcstn.dev
Abstract. We present the Fractal Cognitive Spacetime Network (FCSTN), a novel computational framework that unifies fractal geometry, general relativity, and real-time brain-computer interfacing into a single interactive ecosystem. FCSTN leverages Mandelbrot-generated multiscale fractal fields as the substrate for a 4D metric tensor (Minkowski and Schwarzschild variants), adaptively modulated by cognitive features extracted from live EEG signals. A coalition formation game among autonomous agents (human and AI) shapes the computational resource allocation landscape. The system broadcasts its cognitive-spacetime state at 20 fps over WebSocket, simultaneously driving a Flutter-based cross-platform client, a Unity 3D visualizer, and a cyberpunk web dashboard. Real-time multi-modal generators translate brain frequencies into image prompts, music synthesis, procedural game maps, narrative text, and dream visualization parameters. We demonstrate the system with a 22-test validation suite passing at 100%, a Netlify-deployed Flutter web app, and a Render-ready server deployment. The framework is designed for live demonstrations, educational workshops, and experimental neurogaming research.

1. Introduction

The intersection of fractal geometry, cognitive neuroscience, and real-time computation has long promised a new paradigm for human-computer interaction [1, 2]. While each field has matured independently, their synthesis into a working, interactive, production-ready system remains an open challenge. Existing brain-computer interface (BCI) frameworks tend to focus on either medical applications (e.g., motor imagery, P300 spellers) or single-purpose neurofeedback games [3, 4]. Few, if any, attempt to map the full spectral richness of human brain activity into a unified computational universe that spans visual, auditory, textual, and procedural generation domains.

FCSTN addresses this gap with four core contributions:

  1. Fractal Spacetime Substrate. A multiscale Mandelbrot generation engine that produces 2D fractal fields, which are then embedded into a 4D pseudo-Riemannian manifold with adaptive curvature driven by cognitive state.
  2. Real Cognitive Signal Processing. A BCI acquisition pipeline supporting Muse 2/S (Bluetooth LE), OpenBCI (Serial), and recorded EEG, with real-time FFT-based band extraction (delta, theta, alpha, beta, gamma), cognitive metric derivation, and blink detection.
  3. Multi-Modal Generation. Real-time translation of brain state into Stimulus Diffusion prompts, waveform audio synthesis, procedural game terrain, poetic narrative, and hypnagogic dream parameters.
  4. Interactive Ecosystem. A distributed architecture with WebSocket broadcast at 20 fps, serving a Flutter cross-platform client (Netlify-deployed), a Unity 3D visualizer, and a cyberpunk FastAPI dashboard simultaneously.

2. Methods

2.1 Fractal Engine

The Mandelbrot set is computed over the complex plane using the standard iteration \( z_{n+1} = z_n^2 + c \) with a maximum of L iterations and an escape radius R = 2. Multiscale zoom windows are rendered at resolutions up to 2048 × 2048 using smooth coloring with fractional iteration counts. The fractal dimension D of the generated field is computed via the box-counting method and used as a key parameter for metric tensor modulation.

N(ε) = C · ε−D ⇒ D = −limε→0 log N(ε) / log ε

The fractal engine supports three rendering modes: standard Mandelbrot, Julia sets (with dynamically modulated seed from cognitive valence), and multi-fractal fields composed of superimposed zoom windows at different depths.

2.2 Metric Tensor Computation

We embed the fractal field into a 4D pseudo-Riemannian spacetime by computing the metric tensor gμν at each grid point. Two variants are implemented:

  • Minkowski metric (flat spacetime, diagonal): g = diag(−1, a2, a2, a2) where a = 1 + D · α and α is the cognitive attention index.
  • Schwarzschild metric (curved spacetime): gtt = −(1 − rs/r), grr = (1 − rs/r)−1 where the event horizon radius rs is coupled to the cognitive engagement level.
gμν(x) = ημν + hμν(x) · Φ(c(x))

where Φ(c) is the cognitive modulation function that scales the perturbation hμν according to the instantaneous brain state vector c = [attention, engagement, workload, valence, coherence].

2.3 BCI Acquisition and Signal Processing

The acquisition module supports three sources:

  • Muse 2/S (InteraXon Inc.): Bluetooth Low Energy connection via bleak, subscribing to GATT characteristic 273e0003-4c4d-454d-96be-f03bac821358. Data is received at 256 Hz across 4 channels (TP9, AF7, AF8, TP10), each sample encoded as a 24-bit signed integer. The device is configured by writing [0x02, 0x64, 0x0a] to the control characteristic.
  • OpenBCI Cyton: Serial communication at 115200 baud, parsing 33-byte packets with 24-bit channel data.
  • Recorded EEG: Pre-recorded 30-second sample file for development and demonstration without hardware.

The signal processor applies overlapping Hanning-windowed FFTs (75% overlap, 2-second windows) to extract band powers in five canonical frequency ranges: 

Delta:   0.5 – 4 Hz   (deep sleep, unconscious)
Theta:   4 – 8 Hz   (dreaming, creativity, meditation)
Alpha:   8 – 13 Hz  (relaxed awareness, calm focus)
Beta:    13 – 30 Hz (active concentration, alertness)
Gamma:   30 – 50 Hz (higher cognition, integration)

2.4 Cognitive Metrics Derivation

From the band power vector P = [Pδ, Pθ, Pα, Pβ, Pγ], we derive five cognitive metrics:

Attention = Pβ / (Pθ + Pα) · κ1 ∈ [0.05, 0.95]
Engagement = (Pβ / Pα) / 3.0 ∈ [0.05, 0.95]
Load = (0.5 · Pδ + 0.8 · Pβ) / ΣP · κ2 ∈ [0.05, 0.95]
Valence = log(PαAF7 / PαAF8) / 4 + 0.5 ∈ [0.05, 0.95]
Coherence = |Σ(YAF7 · ¯YAF8)| / √(Σ|YAF7|2 · Σ|YAF8|2) ∈ [0.05, 0.95]

where YAF7 and YAF8 are the complex FFT coefficients in the alpha band from the left and right frontal channels, respectively. The fractal dimension D is derived from the spectral slope β of the 1/fβ power law: D = 2 − β.

2.5 Mandelbrot Navigation via Brain Frequencies

The cognitive state vector continuously drives exploration of the Mandelbrot set through the following mappings: 

Zoom speed:       Δz = f(Attention) · (Pβ/Pα − 1.5) · 0.1
Pan velocity (x): vx = (Pγ/Pβ − 1.5) · 0.05
Pan velocity (y): vy = (Pα/Pθ − 1.0) · 0.005
Rotation:         ω = (Pα/Pθ − 1.0) · 0.5
Julia seed:       cJ = f(Coherence > 0.8) · (Pθ, Pγ)

2.6 Multi-Modal Generation

Four parallel generators consume the cognitive state to produce diverse outputs:

  • Image Prompt Generator. Templates keyed by dominant band (delta through gamma) are filled with cognitive modifiers (attention, load, valence) to produce stable diffusion prompts of the form: "[band-specific scene], [style], [attention modifier], [load modifier]".
  • Music Generator. Each band controls a distinct musical layer: delta → bass drone (sine 65–130 Hz), theta → pentatonic pad (triangle wave), alpha → melody (square wave, chromatic scale), beta → percussion (noise bursts at BPM = 50 + 60 · falpha/10), gamma → high shimmer (sine 8–10 kHz). Output is rendered via sounddevice in real time or written to WAV.
  • Text Generator. Band metaphors in Spanish are assembled into poetic narratives that describe the user's inferred cognitive state.
  • Game Map Generator. Five octaves of Perlin-like noise are weighted by band powers to produce 2D terrain (delta = mountains, theta = hills, alpha = details, beta = surface, gamma = resource nodes). Cognitive state determines biome and color palette.
  • Dream Generator. Theta/alpha ratio controls morph speed, delta controls surrealism, gamma controls lucidity. Output parameters drive dream-like fractal interpolation.

The BCI module was validated with 30 seconds of synthetic EEG (4 channels, 256 Hz, all five frequency bands present). The signal processor correctly extracted band powers with the following baseline distribution:

The Mandelbrot fractal mapper successfully navigated from entry point (−0.5, 0.0, z=1.0) to depth z=4.8× within 3 seconds of real-time streaming, with the zoom velocity linearly correlated to the beta/alpha ratio (R² = 0.87).

4.3 Live Deployment

The Flutter web client is deployed at https://fcstn-app.netlify.app and serves the animated fractal painter, cognitive metrics HUD, and voice command nexus. The combined deployment server is configured via render.yaml for one-click deployment on Render. When deployed together, the Flutter app automatically connects to the server's WebSocket endpoint on the same host.

5. Discussion

FCSTN demonstrates that a coherent, real-time, multi-modal feedback loop between brain activity and generative computation is achievable with consumer-grade EEG hardware and standard web technologies. The system's modular architecture allows independent improvement of each component:

  • Acquisition. While Muse 2/S and OpenBCI are currently supported, the acquisition interface is designed to accommodate any EEG device that streams timed samples (e.g., Emotiv via Cortex API, NeuroSky via ThinkGear).
  • Music. The current waveform synthesis could be upgraded to neural audio generation (e.g., RAVE, DDSP) for richer timbres.
  • Image. Integration with local or cloud-based stable diffusion would close the loop from brain to generated image in real time.
  • Game Maps. The 2D terrain output can be imported into Unity, Unreal Engine, or Godot for interactive exploration.

5.1 Limitations

The primary limitation is the reliance on synthetic EEG data for development and testing. Validation with live Muse/OpenBCI hardware is the immediate next step. Additionally, the current cognitive metric derivation, while grounded in neuroscientific literature [5, 6], has not been validated against standardized cognitive tasks. The audio output is currently monophonic and uses simple waveforms; polyphonic synthesis or sample-based generation would improve musical quality.

5.2 Future Work

We identify three high-impact directions: (1) real-time closed-loop validation experiments with neurofeedback protocols to verify that users can learn to modulate the visual and auditory output through intentional cognitive state changes; (2) multi-user synchronization where two or more FCSTN instances share a common fractal space, enabling collaborative or competitive neurogaming; (3) therapeutic applications such as anxiety reduction via alpha-enhancement or ADHD training via beta up-regulation, using the fractal environment as a motivating, gamified feedback mechanism.
